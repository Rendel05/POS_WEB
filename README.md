Tienda Olly – Panel de Administración Web

Descripción:
Proyecto web de gestión de productos y ofertas para una tienda, desarrollado en PHP y MySQL, con un panel de administración seguro y funcional.

Características principales:

Panel de administración con login protegido solo para administradores.

Creación, edición y desactivación de productos y ofertas.

Formularios dinámicos que muestran el precio regular del producto al crear o editar ofertas.

Gestión de imágenes con nombres únicos basados en el ID del producto y timestamp.

Validación de roles para proteger el acceso a funciones sensibles.

Footer y cabecera consistentes en todo el panel de administración.
